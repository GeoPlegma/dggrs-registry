DGGRS Implement Registry

Abstract

This is a registry for software systems implementing Discrete Global Grid Reference Systems (DGGRS). Each individual in the registry identifies both a software system and DGGRS. Check the DGGRS registry for details on each individual reference system. This registry was created to support the development of the GeoPlegma project, providing unique and universal identifiers for DGGRS and related software.

Introduction back to ToC

Contains a list of individuals of type dggrs:ImplDGGRS. As a Semantic Web document, it provides unique and universal identifiers to each combination of software and DGGRS catalogued.
