Abstract

An ontology for Discrete Global Grid Reference Systems (DGGRS). A DGGRS couples a Discrete Global Grid System (DGGS) tesselating the Earth's surface, with a cell indexing scheme. Thus a DGGRS is able to uniquely and unequivocally identify any position on globe, int theory at whatever resolution desired. The DGGRS ontology was developed as part of the GeoPlegma project. It supports a web registry of DGGRS and the different software systems implementing them.

Introduction back to ToC

Defines classes for DGGRS, Indexing mechanism, Polyhdron and DGGRS implementation, plus the relations between them. Catalogues individuals for Polyhedron and Indexing.
