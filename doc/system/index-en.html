DGGRS Registry

Abstract

This a registry for Discrete Global Grid Reference Systems (DGGRS). Each individual may identify a base polyhedron, a map projection, a base cell shape and an aperture. A DGGRS may be implemented differently in different software packages. Check also the DGGRS implement registry for a thorough list of DGGRS software packages. This registry was created to support the development of the GeoPlegma project, providing unique and universal identifiers for DGGRS and related software.
