DGGRS Registry and Ontology



Overview

An ontology for Discrete Global Grid Reference Systems (DGGRS). A DGGRS couples a Discrete Global Grid System (DGGS) tesselating the Earth's surface, with a cell indexing scheme. Thus a DGGRS is able to uniquely and unequivocally identify any position on the globe, in theory at whatever resolution desired.

This DGGRS ontology was developed as part of the GeoPlegma project. It supports a web registry of DGGRS and the different software systems implementing them. Its main goal is to universally identify reference systems and software DGGS, providing a means of interoperability. These identifiers can be used in datasets, for instance to acurate inform on the underlying topology and reference system.

The assets in this repository are:
  • DGGRS Ontology: formalising the structure of the registry.
  • DGGRS Registry: providing unique identifiers and semantics for DGGRS definitions.
  • Implement Registry: unique identifiers for software implementation of DGGRS.
